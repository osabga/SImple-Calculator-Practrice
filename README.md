# Simple Terminal Calculator

A simple calculator application built for testing the functionality of Visual Studio as a new IDE. This app performs basic arithmetic operations and runs in the terminal.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Requirements
- Visual Studio IDE
- A terminal
